# financial_goals_tracker

# Task Description:
Financial Goal Detail Screen with Firebase Firestore Integration
# MAIN FEATURES OF Finacial goal tracker:
1. Manage and keep track of your financial goals.
2. Keep track of your Expenses with list and pictorial presentations.
3. Feature of adding monthly salary and finacial goals with user frindly UI.
4. All information can be uploaded and fetched from REALTIME FIREBASE DATABASE.
5. Contribution history.
